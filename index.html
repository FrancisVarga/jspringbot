Overview

JSpringBot is a test automation framework for end-to-end acceptance testing and acceptance-test-driven development (ATDD) built on top of Robot Framework using Java, Spring and Maven

Why did I created JSpringBot Framework

  • to have a pure java based library
  • to have a rich library accessible to me as java developer
  • for simple creation of dynamic libraries integrated in Spring and Maven
